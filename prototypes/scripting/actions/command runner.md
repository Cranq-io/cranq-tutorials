# Command runner

[scripting/actions]

Runs an OS command.

### Input ports:

* __state__: _any_

    Receives script state.



* __params__: _{"cwd" :(string or number)[], "result-path" :string, "message" :string, "command" :string}_



### Output ports:

* __state__: _any_

    Sends updated script state.
