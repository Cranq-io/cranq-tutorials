# Action maintainer

### Input ports:

* __action result__: ` any `


* __params__: 
    ```
    {"cwd" :string, "result-path" :string, "message" :string}
    ```


* __state__: ` any `

    Receives script state.

### Output ports:

* __state__: ` any `

    Sends script state as it was written.
