# Adder

[number]

Performs an addition operation on the input values.

Example:

1. 19@0 is received via `a`
2. 2@0 is received via `b`
3. 21@0 is sent via `sum`

### Input ports:

* __a__: _number_

    The first operand



* __b__: _number_

    Second operand



### Output ports:

* __sum__: _number_

    The sum of the two operands.
