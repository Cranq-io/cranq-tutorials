# Even tester

Tells whether a number is even.

Example:

1. 26@0 is received via `number`
2. true@0 is sent via `even`

### Input ports:

* __number__: `number`

    Number to be tested for evenness.

### Output ports:

* __even__: `boolean`

    Sends true if the input value is even, false otherwise.
