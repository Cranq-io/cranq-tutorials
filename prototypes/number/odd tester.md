# Odd tester

[number]

Tells whether a number is odd.

Example:

1. 23@0 is received via `number`
2. true@0 is sent via `odd`

### Input ports:

* __number__: `number`

    Number to be tested for oddness.

### Output ports:

* __odd__: `boolean`

    Sends true if the number is odd, false otherwise.
