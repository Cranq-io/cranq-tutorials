# Query param builder

[io/http/utils]

Builds a single query param specified by a param name and a param value.

Example:
1. 

### Input ports:

* __name__: `string`

    Receives the name of the query parameter.


* __value__: `string`

    Receives the value of the query parameter.

### Output ports:

* __param__: `string`

    Sends the serialized query parameter.
