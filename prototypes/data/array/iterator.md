# Iterator

[data/array]

Iterates through items of an array.


Examples:
https://github.com/Cranq-io/cranq-tutorials/tree/main/reference/1_application_flow/1_2_iterators

### Input ports:

* __array__: _any[]_

    Recieves array to be iterated over.
    
    Example:
    [1,2,3]
    



### Output ports:

* __item__: _any[][number]_

    Sends current item.
    
    Example:
    1



* __index__: _number_

    Sends current index.
    
    Example:
    0



* __done__: _any[]_
