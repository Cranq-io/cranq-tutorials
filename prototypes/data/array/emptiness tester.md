# Emptiness tester

Tests whether array is empty.

Example:
1. []@0 received via `array` 
2. true@0 sent via `empyty`

### Input ports:

* __array__: `any[]`

    Receives array to be tested.
    
    Example:
    []

### Output ports:

* __empty__: `boolean`

    Sends whether the `array` does not contain any element.
    
    Example:
    true
