# Order reverser

Reverses the order of items in the received array.

Example:
1. [1,2,3]@0 received via `array`
2. [3,2,1]@0 sent via `reversed`

### Input ports:

* __array__: `any[]`

    Recieves array to reserve.
    
    Example:
    [1,2,3]

### Output ports:

* __reversed__: `any[]`

    Sends reversed array.
    
    Example:
    [3,2,1
