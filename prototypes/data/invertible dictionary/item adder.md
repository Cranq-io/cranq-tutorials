# Item adder

[data/invertible dictionary]

Adds an item to an invertible dictionary. Invertible dictionaries can have multiple values assigned to a single key.

### Input ports:

* __dict__: `{string: any[]}`


* __key__: `string`


* __value__: `{string: any[]}[string][number]`

### Output ports:

* __dict__: `{string: any[]}`
