# Email signup

Signs up a Firebase user by their email & password.

Email authentication must be enabled for the relevant Firebase Project.

### Input ports:

* __API key__: _any_



* __email__: _any_



* __password__: _any_



### Output ports:

* __ID token__: _any_



* __user ID__: _any_



* __error__: _any_



* __auth context__: _any_
