# Email signup

Signs up a Firebase user by their email & password.

Email authentication must be enabled for the relevant Firebase Project.

### Input ports:

* __API key__: `any`


* __email__: `any`


* __password__: `any`

### Output ports:

* __ID token__: `any`


* __user ID__: `any`


* __error__: `any`


* __auth context__: `any`
