# Lowercaser

[string]

Converts `string`  to lower case.

Example:
1. "Foo"@0  received via `string`
2. "Foo"@0  sent via `lowercase`

### Input ports:

* __string__: _string_

    Receives string to be converted to lower case.
    
    Example:
    "Foo"



### Output ports:

* __lowercase__: _string_

    Sends `string` converted to lower case.
    
    Example:
    "foo"
