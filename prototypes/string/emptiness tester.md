# Emptiness tester

[string]

Tells whether the value received on `string` is empty.

Example:

1. ""@0 is received via `string`
2. true@0 is sent via `empty`

### Input ports:

* __string__: _string_

    The string to be tested for emptiness.



### Output ports:

* __empty__: _boolean_

    Sends true if the string is empty, false otherwise.
