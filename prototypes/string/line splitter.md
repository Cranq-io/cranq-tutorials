# Line splitter

[string]

Splits a string by lines.

Example:
1. 'string' receives 
"line1
line2
line3"
2. 'lines' emits
["line1","line2","line3"]


### Input ports:

* __string__: `string`
    Receives the string to split by lines.
    
    Example:
    "line1
    line2
    line3"



### Output ports:

* __lines__: `string[]`
    Emits the array of lines.
    
    Example:
    ["line1","line2","line3"]
