# Opener

[db/sqlite3]

Opens the specified SQLite database.
Mode is normally 6.
(More on modes: https://www.sqlite.org/c3ref/c_open_autoproxy.html)

### Input ports:

* __params__: _{"dbId" :string, "filename" :string, "mode" :number}_



### Output ports:

* __done__: _null_



* __error__: _string_
