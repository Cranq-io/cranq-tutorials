# Opener (deprecated)

[db/sqlite3]

Opens the specified SQLite database.
Mode is normally 6.
(More on modes: https://www.sqlite.org/c3ref/c_open_autoproxy.html)

### Input ports:

* __DB ID__: _any_



* __file name__: _any_



* __mode__: _any_



### Output ports:

* __done__: _any_



* __error__: _any_



* __DB ID__: _any_
