# Opener (deprecated)

[db/sqlite3]

Opens the specified SQLite database.
Mode is normally 6.
(More on modes: https://www.sqlite.org/c3ref/c_open_autoproxy.html)

### Input ports:

* __DB ID__: `any`


* __file name__: `any`


* __mode__: `any`

### Output ports:

* __done__: `any`


* __error__: `any`


* __DB ID__: `any`
