# Rows getter

[db/sqlite3]

Retrieves multiple rows as a single array from the specified database.

### Input ports:

* __params__: _{"dbId" :string, "sql" :string, optional "params" :(string or number)[]}_



### Output ports:

* __data__: _{string: any}[]_



* __done__: _null_



* __error__: _string_
