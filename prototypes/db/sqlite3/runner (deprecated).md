# Runner (deprecated)

[db/sqlite3]

Runs a single SQL statement on the specified SQLite database, with the specified parameters.

### Input ports:

* __DB ID__: `any`


* __SQL__: `any`


* __params__: `any`


### Output ports:

* __done__: `any`


* __last ID__: `any`


* __changes__: `any`


* __error__: `any`


* __DB ID__: `any`
