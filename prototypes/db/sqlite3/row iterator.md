# Row iterator

Retrieves multiple rows one by one from the specified database.

### Input ports:

* __DB ID__: ` any `


* __SQL__: ` any `


* __params__: ` any `

### Output ports:

* __row__: ` any `


* __done__: ` any `


* __error__: ` any `


* __DB ID__: ` any `
