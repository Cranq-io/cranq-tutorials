# Assertion builder

Builds an assertion record.

### Input ports:

* __success__: `any`

    Receives individual fields for syncing.


* __actual__: `any`

    Receives individual fields for syncing.


* __expected__: `any`

    Receives individual fields for syncing.


* __type__: `any`

### Output ports:

* __assertion__: 
    ```
    {"success" :any, "actual" :any, "expected" :any, "type" :any}
    ```
