# Hardhat config file creator

[blockchain/ethereum/actions]

* __state__: _any_

    Receives script state.



* __params__: _{"cwd" :string, "path" :string, "network" :string, "result-path" :string, "message" :string}_



### Output ports:

* __state__: _any_

    Sends updated script state.
