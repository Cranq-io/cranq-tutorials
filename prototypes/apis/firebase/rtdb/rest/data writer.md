# Data writer

Creates or replaces a data entry  identified by 'path', in a Firebase Realtime Database.

### Input ports:

* __query context__: `any`

    idToken
    dbUrl


* __path__: `any`

    Identifies data entry to be written.
    
    type: string[]


* __data__: `any`

### Output ports:

* __data__: `any`


* __error__: `any`


* __query context__: `any`

    idToken
    dbUrl
