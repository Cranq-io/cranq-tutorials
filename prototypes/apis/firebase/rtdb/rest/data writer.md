# Data writer

[apis/firebase/rtdb/rest]

Creates or replaces a data entry  identified by 'path', in a Firebase Realtime Database.

### Input ports:

* __query context__: _any_

    idToken
    dbUrl



* __path__: _any_

    Identifies data entry to be written.
    
    type: string[]



* __data__: _any_



### Output ports:

* __data__: _any_



* __error__: _any_



* __query context__: _any_

    idToken
    dbUrl
