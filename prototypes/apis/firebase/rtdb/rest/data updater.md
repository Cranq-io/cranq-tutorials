# Data updater

Updates multiple data entries (fields) under 'path', in a Firebase Realtime Database.

### Input ports:

* __query context__: `any`

    idToken
    dbUrl


* __path__: `any`

    Identifies data entry to be updated.
    
    type: string[]


* __data__: `any`

### Output ports:

* __data__: `any`


* __error__: `any`


* __query context__: `any`

    idToken
    dbUrl
