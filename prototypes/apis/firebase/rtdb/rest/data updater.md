# Data updater

[apis/firebase/rtdb/rest]

Updates multiple data entries (fields) under 'path', in a Firebase Realtime Database.

### Input ports:

* __query context__: _any_

    idToken
    dbUrl



* __path__: _any_

    Identifies data entry to be updated.
    
    type: string[]



* __data__: _any_



### Output ports:

* __data__: _any_



* __error__: _any_



* __query context__: _any_

    idToken
    dbUrl
