# Data reader

Reads the data entry identified by 'path', from a Firebase Realtime Database.

### Input ports:

* __query context__: ` any `

    idToken
    dbUrl


* __path__: ` any `

    Identifies data entry to be read.
    
    type: string[]

### Output ports:

* __data__: ` any `


* __error__: ` any `


* __query context__: ` any `

    idToken
    dbUrl
