# Data reader

[apis/firebase/rtdb/rest]

Reads the data entry identified by 'path', from a Firebase Realtime Database.

### Input ports:

* __query context__: _any_

    idToken
    dbUrl



* __path__: _any_

    Identifies data entry to be read.
    
    type: string[]



### Output ports:

* __data__: _any_



* __error__: _any_



* __query context__: _any_

    idToken
    dbUrl
