# Data remover

[apis/firebase/rtdb/rest]

Removes the data entry identified by 'path', from a Firebase Realtime Database.

### Input ports:

* __query context__: _any_

    idToken
    dbUrl



* __path__: _any_

    Identifies data entry to be deleted.
    
    type: string[]



### Output ports:

* __done__: _any_



* __error__: _any_



* __query context__: _any_

    idToken
    dbUrl
