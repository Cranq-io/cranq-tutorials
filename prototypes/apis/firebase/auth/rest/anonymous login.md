# Anonymous login

Logs in a Firebase user anonymously.

Anonymous authentication must be enabled for the relevant Firebase Project.

### Input ports:

* __API key__: `any`

### Output ports:

* __ID token__: `any`


* __user ID__: `any`


* __error__: `any`


* __auth context__: `any`

    idToken
    email
    refreshToken
    expiresIn
    localId
