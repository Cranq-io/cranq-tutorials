# Anonymous login

[apis/firebase/auth/rest]

Logs in a Firebase user anonymously.

Anonymous authentication must be enabled for the relevant Firebase Project.

### Input ports:

* __API key__: _any_



### Output ports:

* __ID token__: _any_



* __user ID__: _any_



* __error__: _any_



* __auth context__: _any_

    idToken
    email
    refreshToken
    expiresIn
    localId
