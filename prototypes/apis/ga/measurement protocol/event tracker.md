# Event tracker

[apis/ga/measurement protocol]

Tracks events via Google Analytics Measurement Protocol.
https://developers.google.com/analytics/devguides/collection/protocol/v1/devguide

### Input ports:

* __tracking ID__: _any_



* __client ID__: _any_



* __category__: _any_



* __action__: _any_



* __label__: _any_



* __value__: _any_



### Output ports:

* __done__: _any_



* __error__: _any_
