# Event tracker

[apis/ga/measurement protocol]

Tracks events via Google Analytics Measurement Protocol.
https://developers.google.com/analytics/devguides/collection/protocol/v1/devguide

### Input ports:

* __tracking ID__: `any`


* __client ID__: `any`


* __category__: `any`


* __action__: `any`


* __label__: `any`


* __value__: `any`


### Output ports:

* __done__: `any`


* __error__: `any`
