# Equality tester

[boolean]

Checks if boolean `a` is equal to boolean `b`.

Example:

1. false@0 is received via `a`
2. false@0 is received via `b`
3. true@0 is sent via `equal`

### Input ports:

* __a__: `boolean`
    First operand



* __b__: `boolean`
    Second operand



### Output ports:

* __equal__: `boolean`
    Sends true if the provided values are equal, false otherwise.
