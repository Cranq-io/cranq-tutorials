# Not

[boolean]

Performs a boolean NOT operation on the value received via input `a`.

Example:

1. false@0 is received via `a`
2. true@0 is sent via `not`

### Input ports:

* __a__: _boolean_

    The value to be negated.



### Output ports:

* __not__: _boolean_

    Sends the negated value of the input `a`.
